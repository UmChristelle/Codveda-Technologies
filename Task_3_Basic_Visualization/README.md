# Level 1 Task 3: Basic Data Visualization

## Objective

Create clear and well-labeled visualizations to show distributions and relationships in the Iris dataset, then export the charts as image files for reporting.

## Dataset

- Iris dataset
- Source file: `data/iris.csv`

## Visualizations Created

- Bar plot of average petal length by species
- Line chart comparing average feature values by species
- Scatter plot of petal length versus petal width
- Scatter plot of sepal length versus sepal width

## Design Focus

- Added clear chart titles, axis labels, and legends
- Used a consistent color palette across plots
- Saved all figures to the `outputs/` folder
