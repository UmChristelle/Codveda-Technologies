# Level 1 Task 2: Exploratory Data Analysis

## Objective

Perform exploratory data analysis on the Iris dataset to summarize the data, visualize feature distributions, and identify relationships between numeric variables.

## Dataset

- Iris dataset
- Source file: `data/iris.csv`

## Work Completed

- Loaded the dataset with pandas
- Calculated summary statistics including mean, median, mode, and standard deviation
- Reviewed missing values and duplicate rows
- Computed correlations between numerical features
- Created histograms, boxplots, scatter plots, and a correlation heatmap
- Saved visual outputs to the `outputs/` folder

## Key Findings

- The dataset has 150 rows and 5 columns
- Total missing values: 0
- Duplicate rows: 3
- Petal length and petal width have the strongest positive correlation at 0.963
- Sepal width has a weak negative relationship with the other numerical variables
